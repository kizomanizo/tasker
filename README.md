## Description

Tasker is a simple learning app that runs tasks as services that are manageable by an authorized and authenticated user. It is a good application in learning hoe NEST works and fundamentals of services.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
